# SmartRevuer — Умный проверяльщик цифр 🤖

Учебный ИИ-проект: многослойный перцептрон обучается распознавать рукописные цифры 0–9 на встроенном датасете scikit-learn. Разработан как полноценный Python-пакет с тестами, линтером и Makefile.

**Точность на тесте: 96.9%** · 26 тестов · lint-чисто

> Для быстрого знакомства с проектом откройте [презентацию](https://xDeady.github.io/SmartRevuer/presentation.html) — 16 слайдов с архитектурой, кодом и графиками.

---

## Быстрый старт

```bash
git clone <repo>
cd SmartRevuer
make install   # pip install -e ".[dev]"
make run       # python -m smartrevuer
```

При запуске скрипт последовательно выполняет 6 шагов:

1. Загружает датасет и выводит статистику по классам
2. Показывает ASCII-визуализацию цифр в терминале
3. Обучает нейросеть (~5 сек) и выводит точность
4. Анализирует ошибки: матрица, топ-5 путаниц, примеры с ASCII
5. Сохраняет 4 PNG-графика в `outputs/`
6. Выводит итоговый чек-лист критериев успеха

После чек-листа в интерактивном терминале запускается демо-игра.

## Архитектура модулей

### `config.py` — константы

Единственный источник правды для всех параметров:

```python
HIDDEN_LAYERS    = (128, 64)   # скрытые слои нейросети
TEST_SIZE        = 0.2          # доля тестовой выборки
RANDOM_STATE     = 42           # воспроизводимость результатов
MAX_ITER         = 500          # максимум итераций обучения

ASCII_CHARS      = ["░", "▒", "▓", "█"]  # символы яркости пикселя
ASCII_THRESHOLDS = [5, 10, 14]            # пороги (пиксели 0–16)

OUTPUT_DIRS      = [Path("/mnt/user-data/outputs"), Path("outputs")]
```

---

### `data.py` — загрузка и препроцессинг

```python
@dataclass
class DataBundle:
    X_train: np.ndarray   # (1437, 64) — нормализованные признаки
    X_test:  np.ndarray   # (360,  64) — нормализованные признаки
    y_train: np.ndarray   # (1437,) — метки классов
    y_test:  np.ndarray   # (360,)  — метки классов
    scaler:  StandardScaler
    X_orig:  np.ndarray   # исходные пиксели 0–16 (для визуализации)
    y_orig:  np.ndarray

def load_and_split() -> DataBundle: ...
```

**Ключевые решения:**
- `StandardScaler.fit_transform()` вызывается только на `X_train` — нет утечки данных в тест
- `stratify=y` гарантирует равномерное представление всех 10 классов в обеих выборках
- `X_orig` хранит исходные пиксели — нормализованные значения непригодны для ASCII-арта

---

### `model.py` — обучение

```python
@dataclass
class ModelResult:
    model:     MLPClassifier
    y_pred:    np.ndarray   # предсказания на тестовой выборке
    train_acc: float        # точность на обучении
    test_acc:  float        # точность на тесте

def train(data: DataBundle) -> ModelResult: ...
```

**Архитектура нейросети:**

```
Вход (64) ──► ReLU (128) ──► ReLU (64) ──► Softmax (10)
```

- 64 входных нейрона — по одному на каждый пиксель изображения 8×8
- Два скрытых слоя с активацией ReLU
- 10 выходных нейронов — вероятности для цифр 0–9

---

### `visualization.py` — ASCII и ANSI

| Функция | Сигнатура | Назначение |
|---|---|---|
| `ascii_digit` | `(pixels: ndarray) -> str` | Массив (64,) → строка 8×8 из символов ░▒▓█ |
| `color` | `(text: str, code: str) -> str` | Оборачивает текст в ANSI escape-код |
| `progress_bar` | `(value, max_value, width=20) -> str` | Полоска `████░░░ 60%` |
| `section_header` | `(step: int, title: str, icon: str) -> str` | Заголовок `─── 🧠 ШАГ 3: Обучение ───` |

Яркость пикселя (0–16) → ASCII-символ:

| Значение | Символ | Яркость |
|---|---|---|
| 0–4 | `░` | тёмный |
| 5–9 | `▒` | средний |
| 10–13 | `▓` | светлый |
| 14–16 | `█` | яркий |

### `cli.py` — пайплайн

```python
def run_pipeline() -> None:
    data   = load_and_split()          # шаг 1: загрузка + статистика
    _print_step1(data)
    _print_step2(data)                 # шаг 2: ASCII-визуализация 5 примеров
    result = train(data)               # шаг 3: обучение нейросети
    _print_step3(data, result)
    _print_step4(data, result)         # шаг 4: анализ ошибок
    paths  = save_all_plots(           # шаг 5: сохранение PNG
        data, result, get_output_dir()
    )
    _print_step5(paths)
    _print_step6(result, paths)        # шаг 6: итоговый чек-лист
    if sys.stdout.isatty():            # демо — только в живом терминале
        interactive_demo(data, result)
```

**Интерактивное демо** (только при `sys.stdout.isatty() == True`):
- Пользователь вводит цифру 0–9, скрипт берёт случайный пример из датасета
- Показывает ASCII-изображение, предсказание ИИ и топ-3 вероятности
- Ведёт счёт угаданных примеров
- Выход по команде `q`

## Этика и ограничения

- Датасет `sklearn.datasets.load_digits` не содержит персональных данных
- Ошибки классификации намеренно показаны пользователю — ИИ не идеален
- Модель работает только с изображениями 8×8 пикселей из этого датасета
- Точность 96.9% означает ~3 ошибки из 100 — это нормально для подобных задач
