ИИ видит цифру как сетку 8×8 пикселей — 64 числа от 0 до 16. Чем больше число, тем ярче пиксель.

Зачем StandardScaler?
Пиксели 0–16. Нейросеть сходится быстрее, если входные числа нормализованы вокруг нуля. Fit только на train — иначе «утечка» данных.
Зачем stratify=y?
Гарантирует, что в тестовой выборке есть все 10 цифр в равной пропорции.
X_orig — зачем?
ASCII-визуализация и интерактивное демо требуют исходные пиксели (0–16), не нормализованные.

ReLU активация
max(0, x) — «включает» нейрон только при положительном сигнале. Позволяет сети учить нелинейные паттерны.
Softmax выход
Превращает 10 чисел в вероятности (сумма = 100%). ИИ: «Это цифра 4 с вероятностью 97%».
Backpropagation
Ошибки «идут назад» и подстраивают веса. 500 итераций — нейросеть запоминает узоры.

⚙️ model.py — обучение и предсказание

@dataclass
class ModelResult:
    model:     MLPClassifier
    y_pred:    np.ndarray  # предсказания на тесте
    train_acc: float       # точность на train
    test_acc:  float       # точность на test

def train(data: DataBundle) -> ModelResult:
    model = MLPClassifier(
        hidden_layer_sizes=HIDDEN_LAYERS,  # (128, 64)
        activation="relu",
        max_iter=MAX_ITER,                 # 500
        random_state=RANDOM_STATE,         # 42
    )
    model.fit(data.X_train, data.y_train)
    return ModelResult(
        model=model,
        y_pred=model.predict(data.X_test),
        train_acc=model.score(data.X_train, data.y_train),
        test_acc= model.score(data.X_test,  data.y_test),
    )
train_acc vs test_acc
Если train≫test — переобучение («зазубрил»). У нас: train=100%, test=96.9% — небольшая разница, норма для этого датасета.
random_state=42
Гарантирует одинаковый результат при каждом запуске. Без него веса инициализируются случайно.

def ascii_digit(pixels: np.ndarray) -> str:
    grid = pixels.reshape(8, 8)
    lines = []
    for row in grid:
        line = ""
        for val in row:
            if   val < 5:  line += "░"  # тёмный
            elif val < 10: line += "▒"  # средний
            elif val < 14: line += "▓"  # светлый
            else:          line += "█"  # яркий
        lines.append(line)
    return "\n".join(lines)

def progress_bar(value, max_value, width=20) -> str:
    filled = int(width * value / max_value)
    bar = "█"*filled + "░"*(width-filled)
    return f"{bar} {int(100*value/max_value)}%"

def section_header(step, title, icon) -> str:
    return color(f"─── {icon} ШАГ {step}: {title} ───",
                 ANSI_BOLD)

def save_all_plots(data, result, output_dir) -> list[Path]:
    return [
        _plot_digit_examples(data, ...),     # 2 ряда × 10 цифр
        _plot_confusion_matrix(data, ...),   # heatmap ошибок
        _plot_accuracy_per_digit(data, ...),  # bar chart по цифрам
        _plot_network_diagram(...),           # схема нейросети
    ]

def get_output_dir() -> Path:
    for d in OUTPUT_DIRS:          # пробуем /mnt/user-data/outputs/
        try:                        # затем ./outputs/
            d.mkdir(exist_ok=True)
            (d / ".probe").touch(); (d / ".probe").unlink()
            return d
        except PermissionError:
            continue
